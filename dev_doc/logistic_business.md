Functions and features

Client (Cargo Owner) Functions
Clients are users who need to send cargo using trucks or moto.

- Register / Login / Reset Password ✅
- Enter cargo details (type, size in kg or volume, pickup location, destination) ✅
- Get automatic cost estimation ✅
- Select truck based on capacity and availability ✅
- View current and past registered cargos✅
- Cancel pending cargos (before approval)   ✅
- Live GPS tracking of truck on map (real-time)
- See estimated time of arrival (ETA)
- Receive notification when cargo is delivered
- Confirm cargo received (via OTP or digital signature)
- Rate driver & service quality
- Pay online (via flutter wave )
- View invoices or receipts

2. Driver (Truck Operator) Functions

- Register / Login ✅
- Set truck status (Available / On Duty / Unavailable)✅
- Accept or reject assigned cargos
- View assigned cargo details✅
- Start delivery → GPS tracking starts
- Real-time location sharing
- Update cargo delivery status
- Upload delivery confirmation (signature or photo)
- View delivery history

3. Admin Functions

- Approve or reject new truck, client, user, driver registrations
- View all registered clients and drivers
- Manage user roles and permissions
- Add/update/remove trucks
- View all cargos with filters (pending, active, delivered)
- Manually assign trucks to cargos (if needed)
- Cancel or reassign cargos
- Set rates per KM and per KG
- Apply surcharges (e.g., urgent delivery, night delivery)
- Apply promo codes or discounts
- Monitor real-time location of all active deliveries
- Get alerts for delays or route deviations
- Generate reports: deliveries, income, usage per client/truck
- Export data to Excel/PDF

4. Super Admin Functions

- Manage all users (admins, clients, drivers)
- Create/edit/delete admin accounts
- Set global pricing policies
- Access full activity logs
- Configure SMS/email notifications
- View system-wide statistics (revenue, usage trends)
- Monitor admin performance
- Dashboard overview of all ongoing operations
- Set role-based access control (RBAC)
- View login logs & location
- System backups and restore

6.Advanced Features

- QR code scan at delivery site
- Multi-language support (English, French, Kinyarwanda, etc.)
