## About the Platform

Loveway Logistics is a comprehensive logistics and cargo delivery platform serving Rwanda. The platform connects clients, drivers, and fleet management with real-time tracking, secure payments, and reliable transportation solutions. We provide fast, secure, and affordable cargo delivery services across Kigali and all districts of Rwanda.

## Technical Stack

**Frontend Framework**: React 18.3.1 with TypeScript
**Build Tool**: Vite 5.4.19
**UI Framework**: 
- shadcn-ui components
- Tailwind CSS 3.4.17
- Radix UI primitives
- Framer Motion for animations
**State Management**: 
- React Query (TanStack Query) for server state
- React Context for global state (Auth, Notifications)
**Routing**: React Router DOM 6.30.1
**HTTP Client**: Axios 1.11.0
**Forms**: React Hook Form with Zod validation
**Maps**: Google Maps JavaScript API
**Real-time**: Socket.IO Client 4.8.1
**Internationalization**: i18next with react-i18next
**Payment**: Flutterwave integration
**Mobile**: Capacitor 7.4.4 (Android support)

## API Integration

**Base URL**: Configured via environment variables
**API Version**: v1
**Authentication**: Bearer token (JWT)
**Key Endpoints**:
- `/v1/auth/*` - Authentication
- `/v1/cargos/*` - Cargo management
- `/v1/tracking/*` - Tracking operations
- `/v1/vehicles/*` - Vehicle management
- `/v1/drivers/*` - Driver management
- `/v1/payments/*` - Payment processing
- `/v1/deliveries/*` - Delivery management
- `/v1/users/*` - User management

**WebSocket**: Real-time updates via Socket.IO
- Connection: `/ws/tracking`
- Events: cargo updates, location updates, status changes

## Important Notes for AI Models

3. **Location Focus**: Services are primarily for Rwanda, especially Kigali
4. **Multi-language**: Content should consider English, Kinyarwanda, and French
